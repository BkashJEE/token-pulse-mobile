# Privacy

Token Pulse Mobile stores only the latest summarized desktop snapshot in a private Vercel Blob. It does not receive raw prompts, message bodies, runtime credentials, desktop files, or the desktop write token through the browser.

The mobile read key is stored in the browser's local storage on the user's device. Users should lock the dashboard before sharing a device and rotate both access keys if a device is lost.

This repository includes no advertising SDK, behavioral analytics, tracking pixel, or third-party crash reporter.
